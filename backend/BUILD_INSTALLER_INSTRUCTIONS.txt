======================================================================
  ColorCorrector - Windows Installer Build Instructions
======================================================================

PREREQUISITES:
--------------
1. Inno Setup 6.x (Download: https://jrsoftware.org/isinfo.php)
   - Use default installation path
   - Free and open source

2. Completed PyInstaller build
   - Must have: backend/dist/ColorCorrector/ folder with all files


STEP-BY-STEP BUILD PROCESS:
----------------------------

Step 1: Build the PyInstaller executable (if not already done)
   Open Command Prompt or PowerShell in the 'backend' folder:
   
   cd backend
   pyinstaller -y colorcorrector.spec
   
   This creates: dist/ColorCorrector/ folder with your app

Step 2: Install Inno Setup (if not already installed)
   - Download from: https://jrsoftware.org/isinfo.php
   - Run the installer
   - Use default installation path
   - No configuration needed

Step 3: Build the Windows installer
   
   Option A - Using the batch script (EASIEST):
   ------------------------------------------
   Double-click: build-installer.bat
   
   OR from command line:
   cd backend
   build-installer.bat
   
   
   Option B - Manual build:
   -----------------------
   1. Open Inno Setup Compiler
   2. File → Open → Select installer-windows.iss
   3. Build → Compile
   4. Wait for completion
   
   
   Option C - Command line:
   -----------------------
   "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer-windows.iss


Step 4: Find your installer
   Location: backend/installer_output/ColorCorrector-Setup-Windows.exe
   Size: ~150-300 MB (includes Python + dependencies)


TESTING THE INSTALLER:
----------------------
1. Run ColorCorrector-Setup-Windows.exe
2. Follow the installation wizard
3. Choose installation directory (default: C:\Program Files\ColorCorrector)
4. Optionally create desktop shortcut
5. Launch the application
6. Browser should open to http://localhost:5000

UNINSTALLING:
-------------
- Control Panel → Programs and Features → ColorCorrector → Uninstall
- OR: Start Menu → ColorCorrector → Uninstall


SYSTEM DEPENDENCIES (Linux AppImage):
--------------------------------------
The Linux AppImage requires the following system libraries to be installed:
- libgl1 (OpenGL library for OpenCV)
- libglib2.0-0 (GLib library)

Install on Ubuntu/Debian:
  sudo apt-get install libgl1 libglib2.0-0

Install on Fedora/RHEL:
  sudo dnf install mesa-libGL glib2

Install on Arch:
  sudo pacman -S mesa glib2

Note: Most desktop Linux distributions have these installed by default.
      Only minimal/server installations may need to install them.


TROUBLESHOOTING:
----------------

Problem: "Inno Setup not found"
Solution: Install from https://jrsoftware.org/isinfo.php
          Or update the path in build-installer.bat

Problem: "dist\ColorCorrector not found"
Solution: Run PyInstaller first: pyinstaller -y colorcorrector.spec

Problem: Installer too large
Solution: This is normal - includes Python runtime and all dependencies
          Typical size: 150-300 MB (Windows), 320-450 MB (macOS/Linux)

Problem: Windows Defender warning
Solution: Normal for unsigned executables. Click "More info" → "Run anyway"
          For production: Sign with a code signing certificate

Problem: Linux AppImage error "libGL.so.1: cannot open shared object file"
Solution: Install OpenGL libraries: sudo apt-get install libgl1 libglib2.0-0

Problem: macOS "App is damaged and can't be opened"
Solution: Run in Terminal: xattr -cr /Applications/ColorCorrector.app
          This removes the quarantine flag from unsigned apps


CUSTOMIZATION:
--------------
Edit installer-windows.iss to customize:
- App name, version, publisher
- Installation defaults
- Desktop icon creation
- License agreement
- Post-install actions
